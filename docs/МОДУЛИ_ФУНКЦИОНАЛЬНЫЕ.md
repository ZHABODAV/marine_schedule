# Функциональные модули системы

Помимо основных логистических модулей, система включает ряд вспомогательных компонентов, обеспечивающих расширенную функциональность.

## 1. Оптимизатор бункера (`bunker_optimizer.py`)

Модуль для расчета и оптимизации затрат на топливо.

* **Функции**:
  * Расчет расхода топлива на рейс (в ходу и в порту).
  * Сравнение стоимости бункеровки в разных портах по маршруту.
  * Рекомендации по месту бункеровки (где дешевле с учетом захода).
* **Входные данные**: Цены на топливо (VLSFO, MGO), профиль потребления судна.

## 2. Контроль доступа (RBAC) (`rbac.py`)

Система управления ролями и правами доступа (Role-Based Access Control).

* **Роли**:
  * `Admin`: Полный доступ.
  * `Manager`: Редактирование планов, но не настроек системы.
  * `Viewer`: Только просмотр отчетов.
* **Функции**: Аутентификация пользователей, проверка прав при доступе к API.

## 3. Генератор PDF отчетов (`pdf_reporter.py`)

Модуль для создания профессиональных печатных отчетов.

* **Возможности**:
  * Генерация `Vessel_Schedule.pdf` с логотипом компании.
  * Создание сводных отчетов для руководства.
  * Поддержка кириллицы и табличного форматирования.

## 4. Валидатор ограничений (`berth_constraints.py`)

Универсальный движок проверки правил, используемый модулем Балаково и другими.

* **Проверки**:
  * Геометрические (длина, ширина, осадка).
  * Временные (пересечение интервалов).
  * Технологические (совместимость грузов, скорость погрузки).

## 5. UI Модули (Frontend)

Набор интерактивных компонентов веб-интерфейса:

* **Alerts Dashboard**: Панель уведомлений о конфликтах и важных событиях.
* **Weather Integration**: Отображение прогноза погоды и штормовых предупреждений.
* **Vessel Tracking**: Визуализация текущего положения судов на карте.
* **Voyage Builder**: Конструктор для ручного создания цепочек рейсов.
