# Модуль Deep Sea (Морские перевозки)

## Обзор
Модуль **Deep Sea** предназначен для планирования и оптимизации международных морских перевозок. Он рассчитывает рейсы океанских судов, учитывая маршруты, проход каналов, погодные условия и характеристики портов.

## Основные функции

### 1. Расчет рейсов (`deepsea_calculator.py`)
*   **Автоматический расчет**: Вычисляет длительность рейса на основе дистанции и скорости судна.
*   **Учет каналов**: Автоматически добавляет время на проход каналов (Суэц, Панама) и ожидание.
*   **Портовые операции**: Рассчитывает время погрузки и выгрузки на основе норм (Load Rate / Discharge Rate) конкретного порта.
*   **Погодный коэффициент**: Добавляет запас времени на погодные условия (Weather Margin).

### 2. Управление сценариями (`deepsea_scenarios.py`)
*   Позволяет создавать альтернативные варианты расписания.
*   Сравнение сценариев по ключевым метрикам:
    *   Общая выручка (Revenue)
    *   Операционные затраты (OPEX)
    *   Прибыль (Profit)
    *   TCE (Time Charter Equivalent)

### 3. Визуализация (`deepsea_gantt_excel.py`)
*   Генерация диаграммы Ганта в Excel.
*   Цветовое кодирование операций:
    *    Погрузка (Loading)
    *    Выгрузка (Discharge)
    *    Морской переход (Sea Transit)
    *    Канал (Canal)
    *    Ожидание (Waiting)

## Входные данные
Модуль использует следующие CSV файлы из папки `input/`:
*   `Vessels.csv`: Характеристики судов (скорость, дедвейт).
*   `Cargo.csv`: Грузы (количество, порты, даты laycan).
*   `Routes.csv`: Матрица расстояний между портами.
*   `Ports.csv`: Параметры портов (нормы погрузки).

## Пример использования
1.  Загрузите данные судов и грузов.
2.  В веб-интерфейсе выберите модуль "Deep Sea".
3.  Нажмите "Сгенерировать расписание".
4.  Скачайте отчет "Диаграмма Ганта" или "Финансовый анализ".
